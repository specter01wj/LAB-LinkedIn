Using the Selectmenu Control

The Selectmenu control provides a highly styleable replacement for the standard HTML <select> control. You specify it in your markup as you usually would a regular select menu, but then call the jQueryUI selectmenu() function on it. The menu continues to work as you would expect in the background, managing its state for form submission and such.
