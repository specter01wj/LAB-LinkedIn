Using the Resizable interaction

The Resizable interaction behavior provides a way to enable page elements to be resized by the user.

The resizing behavior can be configured multiple ways. You can constrain the size of the element, snap to a grid, preserve the aspect ratio, start the resizing behavior only after the mouse has been moved a certain distance, and more.
